# 🗒️ Notes Organizer (Python)
   
A simple **Notes Organizer** created using Python.  
This project allows users to **add, view, search, and delete notes**, saved in a text file.    
     
---          
            
## 🚀 Features     
- **Add Note** – Add a note with title & content 
- **View Notes** – Display all saved notes      
- **Search Note** – Find note by title   
- **Delete Note** – Remove note by title  
- **Simple file handling** (notes saved in notes.txt)          
  
---  

## 🛠️ Technologies Used
- Python
- File Handling
 
---

## 📂 Files Included
- `notes.py` – Main program  
- `notes.txt` – Stores all notes (auto-created)

---

## ▶️ How to Run
1. Download the repository  
2. Open the folder in CMD  
3. Run:

```
python notes.py
```

4. Choose an option from the menu
